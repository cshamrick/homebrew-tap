# homebrew-tap

A [Homebrew](https://brew.sh) tap for [@cshamrick](https://github.com/cshamrick)'s
command-line tools.

## Install

```sh
brew install cshamrick/tap/strobetune
```

Or tap first, then install by name:

```sh
brew tap cshamrick/tap
brew install strobetune
```

## Formulae

- **strobetune** — an analog-style strobe tuner for the terminal
  ([repo](https://github.com/cshamrick/strobetune))
